CodeSmash is a fun and interactive quiz game designed to help you learn the fundamentals of Python. Test your knowledge, earn points, and climb the leaderboard!

## Features

*   Three difficulty levels: Easy, Medium, and Hard.
*   A variety of questions testing your Python knowledge.
*   Score tracking and XP system.
*   A leaderboard to see how you rank against other players.

## How to Play

1.  Open the `index.html` file in your browser.
2.  Select a difficulty level to start the game.
3.  Answer the questions to the best of your ability.
4.  After the quiz, enter your name to be added to the leaderboard.
